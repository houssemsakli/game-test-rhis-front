## Environment
- Java 21
- Maven
- Angular 20
- Node 24.4

## Backend:
mvnw spring-boot:run
(http://localhost:8080)

## Frontend:
npm install
ng serve --open
(http://localhost:4200)

## Usage:
- Register two users (open incognito windows)
- Login both, open lobby, invite one another
- Accept invite -> a game created notification
- Use the board to send moves (prototype minimal)
